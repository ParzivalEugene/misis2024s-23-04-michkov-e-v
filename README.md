## Pre-commit

### Installation

```bash
python3 -m venv ./utils/.venv
. ./utils/.venv/bin/activate
pip install -r ./utils/requirements.txt
```

### Pre-commit hook

```bash
. ./utils/.venv/bin/activate
python3 ./utils/precommit.py

exit_code=$?

if [ $exit_code -ne 0 ]; then
  echo "Error: Python script failed. Commit aborted."
  exit 1
fi

git add .
exit 0
```
